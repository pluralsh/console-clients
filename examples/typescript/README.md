# TypeScript example

Super-simple example that calls the `/me` endpoint.

## Run

1. Build the client (from repo root):

   ```bash
   cd typescript && yarn install && yarn build
   ```

2. Install and run the example:

   ```bash
   cd examples/typescript && yarn install && yarn start
   ```

Set `CONSOLE_URL` and `CONSOLE_TOKEN` via env vars or a `.env` file in this directory (e.g. `CONSOLE_TOKEN=your-token`). `.env` is gitignored.
